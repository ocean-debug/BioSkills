# Changelog

All notable changes to this repository are documented in this file.

## [1.0.0] - 2026-04-24

Initial public BioSkills release.

- Published the first complete canonical `BioSkills` repository to `ocean-debug/BioSkills`.
- Released 157 canonical skills: 130 atomic skills and 27 workflow skills.
- Consolidated content across 5 canonical waves and 31 analysis domains.
- Added generated public artifacts including `skills/catalog.json` and `openclaw.plugin.json`.
- Preserved intake, dedupe, provenance, and Darwin-style scoring outputs under `catalog/` and `darwin/`.
- Included overview diagrams and the grouped input-to-skill-to-workflow dependency tree under `diagrams/`.
- Established MIT licensing for newly written canonical content with provenance-based handling of restricted sources.
